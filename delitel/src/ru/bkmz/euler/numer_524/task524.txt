Рассмотрим следующий алгоритм сортировки списка чисел по возрастанию: Например, список { 4 1 3 2 } будет отсортирован следующим образом: Пусть F(L) будет количеством исполнений шага 2a в процессе сортировки списка L. Например, F({ 4 1 3 2 }) = 5. Мы можем перечислить все перестановки P целых чисел {1, 2, ..., n} в лексикографическом порядке и присвоить каждой перестановке индекс In(P) от 1 до n!, соответствующий ее положению в получившемся списке. Пусть Q(n, k) = min(In(P)) для F(P) = k будет индексом первой перестановки, требующей ровно k шагов для сортировки списка алгоритмом "первая сортировка". Если не существует такой перестановки, для которой F(P) = k, то значение Q(n, k) не определено. Для n = 4 имеем:  Пусть R(k) = min(Q(n, k)) для всех n, для которых значение Q(n, k) определено.
