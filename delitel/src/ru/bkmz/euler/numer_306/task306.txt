Описанная ниже игра представляет собой классический пример теории комбинационных игр: Два игрока начинают с полоской из n белых квадратиков, и совершают ходы по очереди. На каждом ходу, игрок выбирает два следующих друг за другом квадрата и закрашивает их черным цветом. Проигрывает тот игрок, который не может завершить свой ход. Таким образом, при 1 ≤ n ≤ 5, существуют 3 значения n, при которых первый игрок может свести результат к своей победе. Аналогично, при 1 ≤ n ≤ 50, существуют 40 значений n, при которых первый игрок может свести результат к своей победе. Если 1 ≤ n ≤ 1 000 000, то при скольких значениях n первый игрок может свести результат к своей победе?
